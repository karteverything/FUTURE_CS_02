## Tools Used
This document outlines the tools used during the Phishing Email Detection and Awareness
Assessment conducted as part of the Future Interns Cyber Security Internship Task 2.

### Tool Overview
| Tool | Purpose |
|------|---------|
| Public Phishing Email Samples | Source of phishing email examples for analysis |
| Google Message Header Analyzer | Email header inspection and analysis |
| MXToolbox Email Header Analyzer | Sender verification and header analysis |
| Web Browser Tools | Inspection of links, domains, and URLs |
| Microsoft Word | Report creation and documentation |

### The tools used in this assessment provided valuable insight into how phishing 
attacks are structured and how users can identify common warning signs before becoming
victims of email-based cyber threats.
